# 🌍 Energy Access in Africa Dashboard

## Overview
This project analyzes electricity access trends across 54 African countries from 2014–2023 using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Features
- KPI Cards
- Trend Analysis
- Regional Comparison
- Top & Bottom Countries

## Key Insights
- Electricity access improved across Africa over the decade.
- Significant regional disparities still exist.
- North Africa recorded the highest average access levels.
- The analysis highlights strong opportunities for investment in renewable energy infrastructure and rural electrification initiatives.
